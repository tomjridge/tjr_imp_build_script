Module Tjr_pcache.Dcl_example

An example use of the detachable chunked list, for an on-disk persistent cache, with key type int, value type int

For this example, we use simple marshalling based on OCaml's inbuilt module

Each node is stored in a block on disk. Each node contains a list of kv op. We handle marshalling in the write_node parameter
