Module Tjr_pcache.Persistent_list

include Pl_types
type ('ptr, 'a) pl_node = {
next : 'ptr option;
contents : 'a;
}

The type of persistent list nodes with value'a for each node; a singly-linked list.

Nodes in the list have an optional next pointer, and contents.

type ('ptr, 'a) pl_state = {
current_ptr : 'ptr;
current_node : ('ptr, 'a) pl_node;
}

The persistent list state. Consists of current_ptr, a pointer to a block that is currently being written, and current_node, the abstract representation of the contents of the node.

type ('a, 'ptr, 't) pl_ops = {
replace_last : 'a ‑> (unit, 't) Tjr_monad.Types.m;
new_node : 'a ‑> ('ptr, 't) Tjr_monad.Types.m;
}

The operations provided by the persistent list. replace_last replaces the contents of the last element of the list. new_node allocates a new node at the end of the list and makes it the "current" node.

val make_persistent_list : monad_ops:'t Tjr_monad.Types.monad_ops ‑> write_node:('ptr ‑> ('ptr, 'a) pl_node ‑> (unit, 't) Tjr_monad.Types.m) ‑> alloc:(unit ‑> ('ptr, 't) Tjr_monad.Types.m) ‑> with_pl:(('ptr, 'a) pl_state, 't) Tjr_monad.With_state.with_state ‑> ('a, 'ptr, 't) pl_ops

Make a persistent list. Parameters:

  • write_node Write a node to disk, given a ptr to the block and the list_node; this is an operation provided by the lower layer, and may take a long time to complete; assumed synchronous ie doesn't return till data is on disk
  • alloc Allocate a new block; likely provided by pl, in conjunction with a sysytem free list
